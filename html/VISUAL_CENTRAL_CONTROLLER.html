CENTRAL CONTROLLER

Wedge surface — faders, strips, 6-DOF knob, monitors

13 motorized faders: 12 line up with the analog output / summing and monitoring section; 1 master. Faders track the DAW. Above each fader: a motorized encoder for the 12 outputs and pan (plus the master strip as designed). Push an encoder to choose the send layer: e.g. push encoder 1 — all sends use send 1; push encoder 2 — all sends use send 2, and so on.

Digital control, analog path: Touch and 6-DOF knob are digital control for analog components — not a “firmware feature list.” The audio chain stays analog; the screens set what the hardware does.

Parameter focus: When you pick a stage on the main touch grid, detail concentrates on the smaller touchscreen to the right (above the control knob). The gold 6-DOF knob is SpaceMouse-style parameter control for whatever stage is armed.

Face & materials (same line as racks)

Chassisbacklit · acrylic · gold
Enclosure
Backlit LED · black acrylic · see-through — same language as the left and right rack units.
Hardware
Gold-finished knobs and the parameter 6-DOF knob (polished gold / brass family).

Main touchscreen (above faders & encoders)

12 columnsone touch = channel + stage
What it does
This touch area selects which part of the analog chain you’re editing, per channel, in stack order: REC · mic pre · harmonics · compressor · equalizer · transient design · effects. Stage definitions match the left rack — not re-described here.
Record location
Record is on the main 12-column touchscreen strip (REC row per channel), not on scribble long-press.

Left edge — transport & automation

Vertical touch strip
Transport
Touchscreen — rewind, fast-forward, stop, play, record (standard transport cluster).
Automation
Icons / modes for automation and routing aids on the same left strip (layout per CAD).

Scribble strip (below faders)

Touch scribble · per-channel
Voice naming
You can name channels by voice — assign / edit scribble labels with voice (in addition to touch).
Bank
Swipe horizontally to move fader banks / layers.
Mute / solo
Single tap — mute. Double tap — solo. Tap again to release (same strip).

Right — focus screen, 6-DOF knob, monitor controls

Small touchscreen · 6-DOF knob · trims · monitor
Focus LCD
Smaller touchscreen above the 6-DOF knob — shows the focused parameters for whichever rack stage you selected on the main grid.
6-DOF knob
Large gold SpaceMouse-style knob — 6‑DOF parameter control; primary edit surface when a stage is armed.
3×3 trims
Top row: 3× headphone trims · Middle row: 3× output trims · Bottom row: 3× input trims.
Main monitor
Large gold monitor level control — tap to mute monitor path, tap again to unmute.
Bottom trio
Three small gold analog knobs: talkback · mute · fine trim.

Controller link

Control transport
Network
Wired network (Cat6/Ethernet) is the primary controller path for control/state sync; no wireless control required.
Runtime role
Controller link carries control/meter/state data; audio stays in rack conversion/analog paths.
