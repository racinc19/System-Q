Enable public Studio URL (GitHub)
=================================

Repo: https://github.com/racinc19/System-Q

1. GitHub → repo → Settings → Pages.
2. Source: Deploy from a branch → Branch **master** → Folder **/ (root)** → Save.
3. Wait ~1 min. Studio page will be at:
   https://racinc19.github.io/System-Q/studio/

File added: studio/index.html and .nojekyll (disables Jekyll so paths work).

Optional: Custom domain studio.aip2075.com → add in Pages settings and DNS CNAME.
